# GlobeWatch (Global Pulse Map)

GlobeWatch is an interactive globe that visualizes real-time global events (political, economic, and disasters). The app includes an on-map marker experience plus floating panels for live feed updates and event/industry exploration.

Created by `Gaurav Karthikeyan` and `Gnanashree P`.

## How to run

```bash
npm install
npm run dev
```
